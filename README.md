## Source Code Pemograman Perangkat Bergerak
`TI-22-PA/PB/KA`

## Software

- Laravel v11 [here] (https://laravel.com/docs/11.x)
- Install Composer [here] (https://getcomposer.org/download/)
- VSCode as IDE Text Editor
- Web Browser

## Create Project

1. Open terminal, entering to drive path that you want to save the project
2. typing `composer create-project laravel/laravel:^11.0 your-project-name`
3. `cd your-project-name`
4. run the server `php artisan serve`

## Cloning from GIT

1. typing `composer install`
2. generating token `php artisan key:generate`
3. run the server `php artisan serve`
4. run the migration `php artisan migrate`
